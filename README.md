### bicycle-manage（自行车购物系统）
    只是一个简单的自行车购物商城 采用mvc模式的自行车购物商城 包含的模块有：
#### 1.用户登录（包括注册）
#### 2.商品管理
#### 3.购物车
#### 4.商品留言
#### 5.团购报名
